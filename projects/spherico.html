Project

Spherico

Low-Cost Spherical Robot Platform

Overview

Spherico is an inexpensive spherical robot built with laser-cut and 3D-printed parts alongside off-the-shelf components.

Driven by Raspberry Pi and ROS, the platform enables rapid prototyping and can be assembled quickly as a robust base for spherical robot control research.
